# BETCASHBACK — Статус проекта (обновлено 2026-03-24)

## Стек

- **Frontend:** Telegram Mini App, vanilla JS, single-file (`frontend/index.html`, ~4500+ строк)
- **Backend:** Node.js + Express (`backend/server.js`, ~2900 строк)
- **БД:** PostgreSQL — 19 таблиц (+ countries, bookmaker_countries), колонки user_last_read_at, base_cashback_pct, country_code
- **Бот:** Telegram Bot API (inline-кнопки, уведомления)
- **Хостинг:** Render (backend), статика (frontend)

## Бизнес-логика

- Кэшбэк: base 5% + бонус за уровень (до +5%, 6 уровней по сумме одобренного кэшбэка)
- Реферальная система: 4 тира (5%/10%/15%/20%) по суммарному cashback рефералов
- Мин. вывод: зависит от валюты (500₽ / 2500₸ / 16 Br / $5)
- Мин. заявка: зависит от валюты (500₽ / 2500₸ / 16 Br / $5)
- Выплаты: USDT TRC-20, USDT TON, TON
- Страны: 9 СНГ, 4 валюты (RUB/KZT/BYN/USD), выбор при онбординге

## Ключевые файлы

- `frontend/index.html` — весь фронтенд (HTML + CSS + JS)
- `backend/server.js` — весь бэкенд (API + бот + админ-логика)
- `backend/admin/index.html` — админ-панель
- `backend/schema.sql` — полная схема БД
- `backend/migrations/` — SQL миграции (13 файлов, до 013_countries)
